      • writeResults

        public static void writeResults(Path dir,
                List<Hazard> batch,
                OpenOption... options)
                         throws IOException
        Write a batch of HazardResults to files in the specified directory, one for each Imt in the batch. See Files.write(Path, Iterable, java.nio.charset.Charset, OpenOption...) for details on options. If no options are specified, the default behavior is to (over)write a new file. In this case a header row will be written as well. Files are encoded as StandardCharsets.US_ASCII, lat and lon values are formatted to 2 decimal places, and curve values are formatted to 8 significant figures.
        Parameters:
        dir - to write to
        batch - of results to write
        options - specifying how the file is opened
        Throws:
        IOException - if a problem is encountered
        See Also:
        Files.write(Path, Iterable, java.nio.charset.Charset, OpenOption...)
