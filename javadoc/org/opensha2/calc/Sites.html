      • fromString

        public static Sites fromString(String s,
               CalcConfig defaults)
        Create an unmodifiable singleton Iterable<Site> from the supplied string. String is expected to have the form:

        "name,lon,lat[,vs30,vsInf[,z1p0,z2p5]]"

        Although the latter 4 fields are optional, if vs30 is defined, so too must vsInf. Likewise, if z1p0 is defined, so too must z2p5.

        Parameters:
        s - String to parse
