Class LocationGrid

  • java.lang.Object
    • org.opensha2.geo.LocationGrid
  • All Implemented Interfaces:
    Iterable<Location>

    public final class LocationGrid
extends Object
implements Iterable<Location>
    An immutable, tabular grid of locations that supplies row and column data as LocationLists.

    Internally, the grid is backed by a Guava ArrayTable.
