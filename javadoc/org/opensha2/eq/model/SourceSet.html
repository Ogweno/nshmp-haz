Interface SourceSet<T extends Source>

  • All Superinterfaces:
    Comparable<SourceSet<T>>, Iterable<T>, Named
    All Known Implementing Classes:
    AreaSourceSet, ClusterSourceSet, FaultSourceSet, GridSourceSet, InterfaceSourceSet, SlabSourceSet, SystemSourceSet

    public interface SourceSet<T extends Source>
extends Named, Iterable<T>, Comparable<SourceSet<T>>
    Wrapper class for groups of Sources of similar SourceType. The use of the word 'Set' in the class name implies the Sources in a SourceSet will be unique, but no steps are taken to enforce this.
    Author:
    Peter Powers
